# Trial Reports

Create one Markdown file per benchmarked RLLM/RAMA experiment under the matching
status folder.

Use `../templates/trial-report.md` as the starting shape for each report.

## Folders

- `active/` - planned or running trials.
- `success/` - accepted trials with measured positive evidence.
- `failed/` - rejected ideas, regressions, crashes, or negative evidence.
- `inconclusive/` - mixed or insufficient evidence.

Update `index.md` whenever a trial is added or moved.
